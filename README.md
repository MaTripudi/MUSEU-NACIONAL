# teatro-nacional

O projeto **MUSEU-NACIONAL** é um site institucional e informativo desenvolvido com tecnologias web fundamentais (**HTML5, CSS3 e JavaScript**). Ele funciona como uma plataforma digital para um museu fictício ou baseado no Museu Nacional, estruturado de forma equilibrada entre as três linguagens (cerca de 33% de cada uma na composição do código).

Abaixo estão os principais pontos destacados na estrutura do projeto:

### 🏛️ Estrutura e Conteúdo do Site

* **Seções Principais:** O layout conta com uma barra de navegação completa que inclui *Home, Exposições, Pesquisa, Acervo, Fotos, Vídeos* e *Contato*.
* **Exposições Destacadas:** Apresenta painéis informativos sobre exibições de relevância histórica e científica, como a de *Paleoarte*, o legado arqueológico de *Teresa Cristina*, a *Megafauna Pré-histórica (Preguiça Gigante e Tigre-Dentes-de-Sabre)* e o emblemático *Crânio de Luzia*.
* **Seção de Acervos:** Divisão organizada em três pilares principais para fins de estudo e preservação: *Acervo Bibliográfico*, *Acervo Documental* (focado em memórias e culturas indígenas) e *Acervo Científico* (fósseis e biodiversidade).
* **Informações Práticas:** Espaço reservado para a venda/informação de ingressos (valores explícitos de R$ 15,00 e R$ 30,00), horários de funcionamento, localização e programas gratuitos.
* **Mídia:** Inclui uma galeria integrada para exibição de fotos e vídeos para tornar a experiência do usuário mais visual e interativa.

---

### 🛠️ Aspectos Técnicos e Desenvolvimento

* **Tecnologias:** Uso de **HTML** para a marcação estrutural, **CSS** para a estilização gráfica e **JavaScript** (`script.js`) para comportamentos dinâmicos e interatividade na página.
* **Hospedagem:** O projeto está publicado e funcional na web através do **GitHub Pages**.
* **Rodapé institucional:** O site traz a assinatura de direitos autorais vinculada à marca/iniciativa **ANBRUMAH**.
